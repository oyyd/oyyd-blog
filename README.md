#oyyd blog

### Environment

node > 4.2.x

###Develop

```
npm run dev
```

or

```
webpack -w
```

###Commands

translate md to html and generate the rss-feed file from posts:

```
gulp gen-list
```

Release:

```
gulp release
```

Develop:

```
node server.js
```
